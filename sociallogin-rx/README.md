# SocialLogin RxJava module

This module depends on [SocialLogin core library][sociallogin-core]

```Groovy
dependencies {
    compile 'com.jetradarmobile:andorid-sociallogin:1.0.0'
}
```

## Install

```Groovy
dependencies {
    compile 'com.jetradarmobile:sociallogin-rx:1.0.0'
}
```
